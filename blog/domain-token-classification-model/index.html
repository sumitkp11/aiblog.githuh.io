Domain specific token classification model

In the following lines, we will see how to fine-tine a pre-trained language model to perform token classification for specific domains. We will develop an NER model that finds disease names in medical disease abstracts.

We are going to use the NCBI dataset, which contains set of 793 PubMed abstracts, annotated by 14 annotators.
